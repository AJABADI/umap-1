Visualizing the MNIST dataset with UMAP

This vignette demonstrates how to apply umap on the MNIST dataset.

The MNIST dataset is available on local dist.

mnist.file = "/science/resources/MNIST/mnist_all.csv.gz"
mnist.data = read.table(mnist.file, header=F, sep=",")
mnist.labels = mnist.data[,1]
mnist.data = mnist.data[, 2:ncol(mnist.data)]

For this vignette, let's focus on a subset of the data

small = 2000
mnist.small.labels = mnist.labels[1:small]
mnist.small.dist = dist(mnist.data[1:small,])

For reference, cmdscale

mnist.cmdscale = cmdscale(mnist.small.dist)

For umap, can work with entire dataset

uconf = umap.defaults
uconf$n.epochs = 200
uconf$n.neighbors = 15
uconf$verbose=TRUE
mnist.umap = umap(mnist.data[1:10000,], uconf)
